This repository contains source code for AI6103 Projec:Introducing Self-Attention Into ResNet
There are 5 python files in this folder.
 
2 files are for MNIST evaluation for ResNet-18 and BotNet-18
resnet_mnist.py
botnet_mhsa_mnist.py

Another 3 files are for CIFAR-10 evaluation for ResNet-18, BotNet-18 and BotNet-18 improved version
resnet_cifar.py
botnet_mhsa_cifar.py
botnet_mhsa_cifar_improved.py

Simply run the files with python to get the results
